# Hand Gesture Recognition using MediaPipe & OpenCV

This project implements a real-time Hand Gesture Recognition system using a webcam.
It detects hand landmarks and classifies simple gestures like Thumbs Up, Fist, and Open Palm.

## Features
- Real-time hand landmark detection
- Gesture classification
- Webcam-based demo

## Technologies Used
- Python
- MediaPipe
- OpenCV

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the program:
   python main.py
